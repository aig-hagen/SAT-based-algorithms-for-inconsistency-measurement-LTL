Improving MCS Enumeration via Caching


This folder contains a linux executable of tool mcscache [1].

The following command shows the different options to run the tool:

  ./mcscache -h


In the experiments in [1], it was run as follows:

  ./mcscache -nagr -v 0 -T 1800 <cnf-file>



References:

[1] Alessandro Previti, Carlos Mencía, Matti Järvisalo, Joao Marques-Silva.
    Improving MCS Enumeration via Caching. SAT 2017: 184-194
